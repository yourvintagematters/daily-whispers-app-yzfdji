
# Daily Whispers - iOS Build Commands

## Prerequisites
1. Install EAS CLI globally:
   npm install -g eas-cli

2. Login to Expo:
   eas login

3. Ensure you're enrolled in Apple Developer Program ($99/year)

## Building for iOS

### Production Build (for App Store)
eas build --platform ios --profile production

### Preview Build (for internal testing)
eas build --platform ios --profile preview

### Development Build (for development)
eas build --platform ios --profile development

## After Build Completes

### Download the .ipa file
- Go to https://expo.dev/accounts/[your-account]/projects/daily-whispers/builds
- Click on the completed build
- Download the .ipa file

### Submit to App Store
eas submit --platform ios --latest

OR manually upload via Xcode:
1. Open Xcode
2. Go to Window > Organizer
3. Drag and drop your .ipa file
4. Click "Distribute App"
5. Follow the prompts to upload to App Store Connect

## Building for Android

### Production AAB (for Google Play Store)
eas build --platform android --profile production-aab

### Production APK (for direct distribution)
eas build --platform android --profile production

## Checking Build Status
eas build:list

## Viewing Build Logs
eas build:view [build-id]

## Important Notes

1. First build will take longer as EAS sets up certificates
2. You'll be prompted to configure Apple credentials on first iOS build
3. Build typically takes 10-20 minutes
4. You'll receive an email when build completes
5. Builds are stored on EAS servers for 30 days

## Troubleshooting

If build fails:
1. Check build logs: eas build:view [build-id]
2. Ensure all dependencies are compatible with Expo 54
3. Verify app.json configuration is correct
4. Check that bundle identifier is unique

## App Store Submission Checklist

Before submitting to App Store:
✓ Update version number in app.json
✓ Ensure STRIPE_MODE is set to "live" for production
✓ Test the app thoroughly
✓ Prepare App Store screenshots
✓ Write app description and keywords
✓ Set pricing (or free)
✓ Configure age rating
✓ Add privacy policy URL
✓ Submit for review
