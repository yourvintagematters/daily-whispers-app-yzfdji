## Emergency Procedures

### If App is Rejected
1. Read rejection reason carefully
2. Fix the issue
3. Increment build number
4. Rebuild and resubmit
5. Respond to reviewer if needed

### If Payment Issues Occur
1. Check Stripe dashboard for errors
2. Review Supabase Edge Function logs
3. Verify webhook is receiving events
4. Check database for payment records
5. Contact Stripe support if needed

### If Critical Bug Found
1. Assess severity
2. Fix bug immediately
3. Test thoroughly
4. Submit expedited review (if available)
5. Communicate with users via app store updates
