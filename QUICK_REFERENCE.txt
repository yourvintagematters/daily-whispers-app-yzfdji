## 🗄️ DATABASE QUERIES

**Check Renewal Reminders Sent**:
```sql
SELECT COUNT(*) FROM payments WHERE renewal_reminder_sent = true;
```

**Find Upcoming Reminders**:
```sql
SELECT * FROM payments 
WHERE renewal_reminder_sent = false 
AND subscription_end_date BETWEEN NOW() AND NOW() + INTERVAL '7 days';
```

**Check Recent Payments**:
```sql
SELECT * FROM payments 
ORDER BY created_at DESC 
LIMIT 10;
```

**Revenue Summary**:
```sql
SELECT 
  COUNT(*) as total_payments,
  SUM(amount) as total_revenue,
  currency
FROM payments 
WHERE status = 'succeeded'
GROUP BY currency;
```

## 🚨 TROUBLESHOOTING

**Emails Not Sending**
1. Check `RESEND_API_KEY` in Supabase secrets
2. View Edge Function logs
3. Verify sender domain in Resend
4. Check recipient email addresses

**Payment Failing**
1. Check Stripe API keys
2. Verify AUD is enabled in Stripe
3. Review payment logs
4. Test with test card

**Cron Not Running**
1. Check GitHub Actions tab
2. Verify cron service status
3. Test manual trigger
4. Check authorization

**Database Issues**
1. Check RLS policies
2. Verify migrations applied
3. Review table structure
4. Check connection
